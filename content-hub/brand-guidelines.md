# Brand Guidelines

> Maintain consistent branding across all platforms and content types.

---

## Brand Identity

### Voice & Tone

| Attribute | Description |
|-----------|-------------|
| **Confident** | Speak with authority, backed by data |
| **Direct** | Short sentences, no fluff |
| **Forward-thinking** | Focus on future implications |
| **Accessible** | Complex topics made simple |
| **Authentic** | Real opinions, hot takes welcome |

### Writing Style

- **Punchy** - One idea per line
- **Scannable** - Headers, bullets, whitespace
- **Data-driven** - Stats and sources
- **Conversational** - Write like you speak
- **Action-oriented** - Clear takeaways

---

## Visual Identity

### Color Palette

| Color | Hex | Usage |
|-------|-----|-------|
| Primary Black | `#0A0A0A` | Headlines, text |
| Electric Blue | `#0066FF` | CTAs, highlights |
| Accent Purple | `#7C3AED` | AI/Tech topics |
| Success Green | `#10B981` | Positive stats |
| Warning Orange | `#F59E0B` | Alerts, trends |
| Clean White | `#FFFFFF` | Backgrounds |

### Typography

| Element | Font | Weight |
|---------|------|--------|
| Headlines | Inter | Bold (700) |
| Body | Inter | Regular (400) |
| Code/Data | JetBrains Mono | Medium (500) |
| Accent | Space Grotesk | Semi-bold (600) |

### Logo Usage

- Minimum size: 32px height
- Clear space: 1x logo height on all sides
- Never stretch, rotate, or add effects

## Platform-Specific Guidelines

### LinkedIn

```
Format:
- Hook in first line (pattern interrupt)
- One sentence per line
- Use --- for section breaks
- End with question/CTA
- 3-5 relevant hashtags

Character limit: 3,000
Optimal length: 1,200-1,500
Best posting times: Tue-Thu, 8-10am
```

### Twitter/X

```
Format:
- Strong opening hook
- Thread if >280 chars
- Use line breaks for readability
- End with engagement hook
- 1-2 hashtags max

Character limit: 280 (or thread)
Best posting times: Mon-Fri, 9am-12pm
```

### YouTube

```
Thumbnail:
- Bold text (3-5 words max)
- High contrast colors
- Face showing emotion
- 1280x720px minimum

Title:
- Under 60 characters
- Include keyword
- Create curiosity

Description:
- First 150 chars = hook
- Timestamps
- Links and CTAs
```

### Instagram

```
Carousel:
- 1080x1350px (4:5 ratio)
- First slide = hook
- Last slide = CTA
- 5-10 slides optimal

Caption:
- Hook in first line
- Story/value in middle
- CTA at end
- Hashtags in comment or end
```

### TikTok

```
Video:
- 1080x1920px (9:16)
- Hook in first 3 seconds
- 30-60 seconds optimal
- Trending sounds when relevant

Caption:
- Short and punchy
- 3-5 hashtags
- Include trending tags
```

## Content Templates

### Post Hook Formulas

1. **Pattern Interrupt**
   ```
   Stop scrolling. [Bold claim].
   ```

2. **Controversial Take**
   ```
   Unpopular opinion: [Statement]
   ```

3. **Data Shock**
   ```
   [Statistic]. Let that sink in.
   ```

4. **Story Open**
   ```
   [Time] ago, I [action]. Here's what happened:
   ```

5. **Question Hook**
   ```
   Why does [thing] when [observation]?
   ```

### CTA Formulas

1. **Engagement Ask**
   ```
   What's your take? Drop it below.
   ```

2. **Share Prompt**
   ```
   Share this with someone who needs to see it.
   ```

3. **Follow Hook**
   ```
   Follow for more [topic] insights.
   ```

4. **Save Prompt**
   ```
   Save this for later. You'll need it.
   ```
